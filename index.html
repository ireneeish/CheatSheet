HTML Tags

Tag Name Description
<body> Body Contains all the visible content of an HTML document
<h1> to <h6> Headings Heading and sub-headings used to enlarge text
<p> Paragraph Contains paragraphs of text
<div> Division Layout your page using divisions or sections
<span> Span Used to group in-line elements in a document
<ol> Ordered List Used for a numbered list
<ul> Unordered List Used for a bulleted list
<br> Break Creates a line break
<head> Head Contains information about the web page, like the title
<title> Title Web page title that will show in browser tab
<a> Anchor Link to other pages or content on the same page

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining the column headers in a table
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row The table row container
<td> Table Data The table data container
<tfoot> Table Footer The set of rows defining the footer in a table

Table Attributes

Tag Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row Span Defines how many rows a td element should span
